## 常用技巧

### 激活虚拟环境

每次进入项目，要激活一下

```
source venv/bin/activate
```

### 安装

Flask的安装，详见[Installation](https://flask.palletsprojects.com/en/1.1.x/installation/#python-version)。

```
virtualenv --no-site-packages venv
source venv/bin/activate
pip install flask
flask --version
```
